get.amino.acids {MSnbase}R Documentation

Amino acids

Description

Returns a data.frame of amino acid properties: AA, ResidueMass, Abbrev3, ImmoniumIonMass, Name, Hydrophobicity, Hydrophilicity, SideChainMass, pK1, pK2 and pI.

Usage

get.amino.acids()

Value

A data.frame

Author(s)

Laurent Gatto

Examples

get.amino.acids()
